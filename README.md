Setup
-----

## Ubuntu

* Install [Ruby-Build](https://github.com/sstephenson/ruby-build)
* Install [rbenv](https://github.com/sstephenson/rbenv)

```bash
$ sudo apt-get install nodejs
$ gem install bundler
$ rbenv rehash
$ bundle install
$ rbenv rehash
$ bundle exec jekyll serve
```
